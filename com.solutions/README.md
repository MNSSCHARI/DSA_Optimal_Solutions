## Solutions Included

### 1. **ArraySorted.java**
   - **Purpose**: Check if an array is sorted in ascending order
   - **Algorithm**: Single pass linear scan
   - **Time Complexity**: O(n)
   - **Space Complexity**: O(1)
   - **Returns**: `true` if array is sorted, `false` otherwise

### 2. **LargeNumArr.java**
   - **Purpose**: Find the largest element in an array
   - **Algorithm**: Linear search with single pass
   - **Time Complexity**: O(n)
   - **Space Complexity**: O(1)
   - **Returns**: The maximum element in the array

### 3. **RemoveDupInArray.java**
   - **Purpose**: Remove duplicates from a sorted array in-place
   - **Algorithm**: Two-pointer approach
   - **Time Complexity**: O(n)
   - **Space Complexity**: O(1)
   - **Returns**: The count of unique elements
   - **Note**: Assumes the input array is sorted

### 4. **SecondLargestArr.java**
   - **Purpose**: Find the second largest element in an array
   - **Algorithm**: Single pass with two variable tracking
   - **Time Complexity**: O(n)
   - **Space Complexity**: O(1)
   - **Returns**: The second largest element, or -1 if no second largest exists
